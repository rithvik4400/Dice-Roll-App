# 🎲 Dice Roll App

## 🚀 Overview

The Dice Roll App is a simple Android application built using Kotlin and Jetpack Compose that simulates rolling a dice. When the user taps a button, the app generates a random number and updates the UI with the corresponding dice image.

---

## ✨ Features

* 🎲 Roll a virtual dice
* 🔢 Generates random numbers (1–6)
* 🖼️ Displays corresponding dice image
* 👆 Button-based interaction
* 📱 Built with Jetpack Compose

---

## 🛠️ Tech Stack

* **Language:** Kotlin
* **Framework:** Android (Jetpack Compose)
* **IDE:** Android Studio

## ⚙️ How It Works

1. User taps the **Roll Dice** button
2. App generates a random number between 1 and 6
3. Corresponding dice image is displayed
4. UI updates instantly

---

## 🎯 Learning Outcomes

* Working with **random number generation**
* Handling button click events
* Updating UI dynamically
* Managing state in Jetpack Compose
